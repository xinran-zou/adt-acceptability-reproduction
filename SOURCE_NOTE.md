# Source Note

This repository provides supplementary reproduction and artifact evaluation materials for the paper *A Limited Technical Background Is Sufficient for Attack-Defense Tree Acceptability*.

The original artifact files were obtained from the authors' Zenodo repository:

https://zenodo.org/records/14717343

The original artifact includes files such as `Statistics.ipynb`, `Survey Data.csv`, ADT archive files, study PDFs, the qualitative evaluation rubric, lecture plan, and lecture slides. These original files are not redistributed in this repository.

This repository only provides supplementary materials prepared for reproduction and artifact evaluation purposes, including `README.md`, `requirements.txt`, and `reported_vs_reproduced_discrepancies.csv`.

The original data, notebooks, study materials, ADT files, PDFs, and slides belong to the original authors. No additional license is applied to the original artifact files by this repository.
